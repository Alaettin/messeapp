Neoception Passport API

Connector API

REST API im Connector-Muster. Alle Endpunkte sind öffentlich (kein Login erforderlich).

Base URL

Endpoints

GET /Product/ids List all visitor IDs

Returns an array of all visitor (item) IDs.

Response 200 OK
["12kjo213asd", "abc456", "xyz789"]
GET /Product/hierarchies List hierarchies (reserved)

Returns an empty array. Reserved for future use.

Response 200 OK
[]
GET /Product/hierarchy/levels List hierarchy levels

Returns one level: "Asset".

Response 200 OK
[
  { "level": 1, "name": "Asset" }
]
GET /model List model datapoints

Returns all defined datapoints including visitor properties, documents, and contacts. Type 0 = Property, Type 1 = File.

Response 200 OK
[
  { "id": "name", "name": "Name", "type": 0 },
  { "id": "company", "name": "Firma", "type": 0 },
  { "id": "avatar", "name": "Avatar", "type": 1 },
  { "id": "document_1", "name": "Produktkatalog", "type": 1 },
  { "id": "contact_1_name", "name": "Ansprechpartner 1 - Name", "type": 0 }
]
GET /Product/{itemId}/hierarchy Get hierarchy values for a visitor

Returns the hierarchy value for a specific visitor.

Response 200 OK
[
  { "level": 1, "name": "Avatar" }
]
Error 404 Not Found
{ "error": "Asset not found" }
POST /Product/{itemId}/values Get property values for a visitor

Returns property values for a specific visitor. Use the request body to filter by property IDs. Send empty arrays to get all values. Files return needsResolve: true — use /documents to resolve them.

Request Body
{
  "propertiesWithLanguage": {
    "languages": [],
    "propertyIds": []
  },
  "propertiesWithoutLanguage": {
    "propertyIds": []
  }
}
Response 200 OK
[
  {
    "propertyId": "name",
    "value": "Max Mustermann",
    "valueLanguage": "en",
    "needsResolve": false
  },
  {
    "propertyId": "avatar",
    "value": "avatar_12kjo213asd",
    "valueLanguage": "en",
    "mimeType": "image/png",
    "filename": "avatar",
    "needsResolve": true
  }
]
Error 404 Not Found
{ "error": "Asset not found" }
POST /Product/{itemId}/documents Resolve file content

Returns the full Base64-encoded file content for the requested document properties. Use this to resolve files that were returned with needsResolve: true in the /values endpoint.

Request Body
{
  "languages": ["en"],
  "propertyIds": ["avatar"]
}
Response 200 OK
[
  {
    "propertyId": "avatar",
    "value": "iVBORw0KGgo...",
    "filename": "avatar",
    "valueLanguage": "en",
    "needsResolve": false
  }
]
Error 404 Not Found
{ "error": "Asset not found" }
